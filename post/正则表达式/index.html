正则表达式

语法

限定符

规则

  • ”+，*，？”含义
    • ”+“，例如”abc+b”可以匹配”abcb”,“abccb”,但是不能匹配”abb”,“+“表示至少出现一次；
    • “*“,例如”abc*b”可以匹配”abb”,“abcb”等，”*“表示可以出现0或者一次以上；
    • ”?“,例如”abc?b”可以匹配”abcb”,“abb”,但是不能匹配”abccb”，”?“表示可以最多出现一次；
    • ”{n}“，例如”ab{2}“只能匹配”abb”,“{n}“表示出现n次；
    • ”{n,}“,例如”ab{2,}“,可以匹配”abb”,“abbb”, “{n,}“表示至少出现n次；
    • ”{m, n}“,其中m <= n,表示最少出现m次，最多出现n次。

总结：

  • ”+“相当于”{1,}” ；
  • “*“相当于”{0,}” ；
  • ”?“相当于”{0,1}” ；
  • 在”*“或者”+“后加”?“将贪婪限定符转为非贪婪 。

定位符

规则

  • ”^“匹配字符串开始的地方
  • ”$“匹配字符串结束的位置
  • “\b”单词边界
  • “\B”非单词边界

常用元字符

  • [xyz],匹配字符集中的任意一个字符
  • [^xyz],匹配除了字符集以外的任何字符
  • x|y，匹配x或者y
  • [a-z],匹配a-z中的任意一个字符
  • [^a-z],匹配任意一个不在a-z中的字符
  • \d,匹配一个数字字符，相当于[0-9]
  • \D,匹配任意一个非数字字符，相当于[^0-9]

正则表达式在java中的应用

类介绍

  • Pattern类，正则表达式的编译表示
String regex = "((19|20)[0-9]{2})([0-9]{9})";
Pattern pattern = Pattern.compile(regex);
  • Matcher类，Matcher 对象是对输入字符串进行解释和匹配操作的引擎。
String regex = "((19|20)[0-9]{2})([0-9]{9})";
Pattern pattern = Pattern.compile(regex);
Matcher matcher = pattern.matcher("2018123456789");
  • find()
    • find(): return boolean,该方法查找下一个匹配正则表达式的子串，如果找到返回true，找不到返回false；
    • find(int start);同上一个方法一样，只是会从start的index开始查找；
  • group()
    • group()：是在查找到匹配正则的表达式的子串之后， 对子串的分解，其中group()返回的是整个子串,group()等价于group(0)；
    • group(int index),返回的匹配正则里的index个正则的部分，比如正则表达式”^(([0-9]{19})(\s))$“中，第一个正则是整个正则部分([0-9]{19})(\s)，然后第二个是[0-9]{19},第三个是\s;这时候是以（）为准的，并且满足先外面后里面的规则，如果正则表达式里没有（）则除了group(0)，没有其他group

一般用法示例

String regex = "((19|20)[0-9]{2})([0-9]{10})";

Pattern pattern = Pattern.compile(regex);

Matcher matcher = pattern.matcher("20181234567890");

int count = 0;
while(matcher.find()) {

    count++;
    System.out.println(matcher.group(0));
    System.out.println(matcher.group(1));
    System.out.println(matcher.group(2));
    System.out.println(matcher.group(3));
}
