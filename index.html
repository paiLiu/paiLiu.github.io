正则表达式 语法 限定符 规则 ”+，*，？”含义 ”+“，例如”abc+b”可以匹配”abcb”,“abccb”,但是不能匹配”abb”,“+“表示至少出现一次； “*“,例如”abc*b”可以匹配”abb”,“abcb”等，”*“表示可以出现0或者一次以上； ”?“,例如”abc?b”可以匹配”abcb”,“abb”,但是不能匹配”abccb”，”?“表示可以最多出现一次； ”{n}“，例如”ab{2}“只能匹配”abb”,“{n}“表示出现n次； ”{n,}“,例如”ab{2,}“,可以匹配”abb”,“abbb”, “{n,}“表示至少出现n次； ”{m, n}“,其中m <= n,表示最少出现m次，最多出现n次。 总结： ”+“相当于”{1,}” ； “*“相当于”{0,}” ； ”?“相当于”{0,1}” ； 在”*“或者”+“后加”?“将贪婪限定符转为非贪婪 。 定位符 规则 ”^“匹配字符串开始的地方 ”$“匹配字符串结束的位置 “\b”单词边界 “\B”非单词边界 常用元字符 [xyz],匹配字符集中的任意一个字符 [^xyz],匹配除了字符集以外的任何字符 x|y，匹配x或者y [a-z],匹配a-z中的任意一个字符 [^a-z],匹配任意一个不在a-z中的字符 \d,匹配一个数字字符，相当于[0-9] \D,匹配任意一个非数字字符，相当于[^0-9] 正则表达式在java中的应用 类介绍 Pattern类，正则表达式的编译表示 String regex = "((19|20)[0-9]{2})([0-9]{9})"; Pattern pattern = Pattern.compile(regex); Matcher类，Matcher 对象是对输入字符串进行解释和匹配操作的引擎。 String regex = "((19|20)[0-9]{2})([0-9]{9})"; Pattern pattern = Pattern.
